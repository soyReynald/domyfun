How To - top

How to Select layers in Figma?

You can use Move Tool (V) for fast layer selection.



How to edit each section in Figma?

Each section is separated with each folder with differ color marks.



How to change images in Figma?

Place the image in image placeholder layer with clipping mask. So fit the image related position within the box.



How to edit text in Figma?

Select the text layer and change the text.



HOW TO EDIT

In order to edit the file, you need to have Figma installed on your system, and the fonts mentioned in thecredits installed. As you can see on the right, the layers are well organized and gropued into folders, which make editing very easy. So, all you have to do, is select the layer from the right, and then edit it with Figma
